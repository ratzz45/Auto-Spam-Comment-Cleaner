Automatically deletes spam comments every day. Simple, lightweight, and hassle-free spam control for your WordPress site.

== Description ==
This plugin runs a daily task that deletes all comments marked as spam from your database. It's fully automatic — just activate and let it clean up for you!

= Features =
* Automatically runs daily via WordPress cron
* Deletes all spam comments
* Lightweight and simple to use

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! Spam comments will now be deleted daily.

== Frequently Asked Questions ==
= Does this plugin have any settings? =
No, it's automatic. Just activate and it will start working.

= Can I change the frequency? =
Currently, it runs once per day. Future versions may allow frequency customization.
